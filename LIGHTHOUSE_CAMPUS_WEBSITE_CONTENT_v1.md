## Page Template (applied to every page below)

Every page in this document uses the following structure:

- **Page Purpose** — why this page exists and what it must accomplish.
- **Hero / Page Header** — eyebrow, headline, supporting line, primary imagery notes.
- **Introduction** — opening narrative paragraphs.
- **Main Sections** — ordered content blocks with headings and body copy.
- **Cards** — repeating card blocks (title + short body, optional icon/image).
- **Quotes** — pull-quotes with attribution.
- **Statistics** — numeric highlights with labels and context.
- **Call to Action** — closing invitation, primary and secondary actions.
- **Related Pages** — internal cross-links.
- **SEO Title** — `<title>` for the route (≤ 60 chars).
- **Meta Description** — description meta (≤ 160 chars).
